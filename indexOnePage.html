Alcance del Proyecto


Objetivo Principal

Nuestro objetivo con este proyecto es poder otorgar, de manera más cómoda y fácil, un servicio de reserva de pasajes para buses interurbanos dentro de una región específica.



Alcance

Funcionalidad de reserva de pasajes de bus para una o varias rutas específicas, lo que permitiría a los usuarios seleccionar su destino y las fecha de viaje deseada. Ademas, una interfaz fácil de usar y atractiva para los usuarios, que permita a los usuarios navegar fácilmente por el sitio web y hacer reservas de manera rápida y sencilla.
